# Etiquetador de Cestas

Esta es una herramienta simple para generar etiquetas para cestas de productos. Permite al usuario especificar el número de etiquetas, el texto de la cesta y cargar un logo personalizado.

## Despliegue

Esta aplicación es un único archivo HTML (`etiquetador.html`) y no requiere un servidor web ni un proceso de construcción. Para "desplegarla", simplemente guarde el archivo en su ordenador.

## Instrucciones de Uso

1.  **Abra el archivo `etiquetador.html`** en cualquier navegador web moderno (como Chrome, Firefox, Edge, etc.). Puede hacerlo haciendo doble clic en el archivo.
2.  **Rellene los campos del formulario:**
    *   **Número de etiquetas:** Introduzca la cantidad de etiquetas que desea imprimir.
    *   **Tipo de cesta:** Escriba el texto que desea que aparezca en la etiqueta (por ejemplo, "Cesta Navideña Premium").
    *   **Logo (opcional):** Haga clic en "Seleccionar archivo" para elegir una imagen de su ordenador que se usará como logo. Si no selecciona ningún archivo, las etiquetas se generarán sin logo.
3.  **Haga clic en el botón "Generar Etiquetas".**
4.  **Verifique la vista previa:** Se mostrará una sola etiqueta en la pantalla para que pueda verificar que el diseño y el texto son correctos.
5.  **Imprima las etiquetas:**
    *   Vaya al menú de su navegador y seleccione **Imprimir** (o use el atajo `Ctrl+P` o `Cmd+P`).
    *   Se imprimirá la cantidad total de etiquetas que especificó, cada una en una página separada, con el formato correcto para una impresora de etiquetas de 70mm x 90mm.

¡Y eso es todo! Ya tiene sus etiquetas listas.
